Chi ha inventato questo linguaggio?

Il linguaggio HTML è stato inventato da Tim Barners-lee


Quando è stato inventato?

E' stato inventato nel 1989


Quali sono gli scopi di questo linguaggio?

HTML è utilizzato per definire la struttura di una pagina web, rendere i contenuti accessibili, separare il contenuto dalla presentazione con CSS e per integrare elementi multimediali facilitando l'interazione con gli utenti attraverso form e altri elementi interattivi


Che cose il WEB?

Il World Wide Web comunemente chiamato web, è un sistema di documenti interconnessi accessibili tramite internet


Che problemi risolveva il WEB ?

Il web è stato creato per risolvere diversi problemi legati alla condivisione e all'accesso alle informazioni


Che differenza tra web e html ed internet?

Internet è una rete globale di computer interconnessi che supporta vari servizi, tra cui il web. Il web è un servizioche utilizza internet per accedere ai documenti e alle risorse collegati tramite link ipertestuali. HTML è il linguaggio usato per creare e strutturare le pagine web
