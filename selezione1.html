SELEZIONE

La selezione o anche detta se condizionale è un'operazione che controlla un test di tipo booleano (cioè se un'espressione dà vero o falso)

Se il test è vero eseguo una sequenza di istruzioni. Se il test è falso eseguo un'altra sequenza di istruzioni.

Attenzione esempio:

if (TEST) restituisce o vero o falso

{ 1 costrutto se test == true eseguo il primo costrutto di istruzioni all'interno delle prime parentesi graffe }

else (altrimenti)

{ 2 costrutto se test == false eseguo il secondo costrutto di istruzioni all'interno delle prime parentesi graffe }
